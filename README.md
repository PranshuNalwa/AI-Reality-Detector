# AI Reality Distortion Detector

A comprehensive Flask web application that helps users detect **AI-generated content**, **deepfakes**, **misinformation**, and **online threats** — all in one unified platform.

Upload an image, paste text, scan a QR code, verify a website, or fact-check a claim — and get instant, actionable results.

---

## Features

### Image Analysis
- **AI & Deepfake Detection** — Upload an image and receive a 0–100 manipulation confidence score powered by Sightengine's `genai` and `deepfake` classifiers.
- **EXIF Metadata Extraction** — Inspect camera make/model, software tags, and date information embedded in the image using Pillow.
- **Dual Explanations** — Every result includes both a plain-language summary for general users and a detailed technical breakdown.

### AI Text Detection
- **AI-Written Content Detector** — Paste any text (50+ characters) and get a probability score indicating whether it was written by a human or generated by AI, powered by the Sapling AI API.

### QR Code Scanner
- **QR Decode & Threat Analysis** — Upload a QR code image to decode its contents. URLs are automatically checked against the Google Safe Browsing API.
- **Payment Scam Detection** — UPI payment QR codes are flagged with payee details so users can verify before transacting.

### Website Safety Checker
- **URL Threat Lookup** — Enter any URL to check it against Google Safe Browsing for malware, social engineering, and unwanted software threats.

### Fact Checker
- **Claim Verification** — Enter a claim or paste a URL to search the Google Fact Check Tools API for existing fact-check reviews from trusted publishers.
- **Real-Time News Scraping** — Aggregates related news articles and cross-references sources to compute a 0–100 credibility score with a verdict (Authentic / Inconclusive / False).

### General
- **Drag-and-Drop Upload** — Modern upload interface with instant file preview.
- **Scan History** — Browse all past image analyses with color-coded score badges.
- **Share Results** — Copy result links to your clipboard with one click.
- **Responsive UI** — Clean, mobile-friendly design.

## Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/AI-Reality-Detector.git
cd AI-Reality-Detector
```

### 2. Create a virtual environment

```bash
python -m venv venv
source venv/bin/activate   # macOS / Linux
venv\Scripts\activate      # Windows
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

> **Note (macOS):** QR scanning requires `zbar`. Install it with:
> ```bash
> brew install zbar
> ```

### 4. Configure environment variables

Create a `.env` file in the project root (or edit the existing one) with your API credentials:

```env
# Flask
SECRET_KEY=your-random-secret-key

# Sightengine — https://sightengine.com
SIGHTENGINE_API_USER=your-api-user
SIGHTENGINE_API_SECRET=your-api-secret

# Google Safe Browsing — https://console.cloud.google.com
GOOGLE_API_KEY=your-google-api-key

# Google Fact Check Tools — https://console.cloud.google.com
FACT_CHECK_API_KEY=your-fact-check-api-key

# Sapling AI — https://sapling.ai
SAPLING_API_KEY=your-sapling-api-key
```

### 5. Run the application

```bash
python run.py
```

Visit **http://127.0.0.1:8000** in your browser.

## API Endpoints

| Method | Endpoint              | Description                          |
|--------|-----------------------|--------------------------------------|
| GET    | `/`                   | Home page — image upload form        |
| POST   | `/`                   | Submit an image for AI analysis      |
| GET    | `/result/<scan_id>`   | View results for a specific scan     |
| GET    | `/history`            | Browse all past scans                |
| POST   | `/api/scan-qr`        | Decode & analyze a QR code image     |
| POST   | `/api/check-text`     | Detect AI-generated text             |
| POST   | `/api/fact-check`     | Fact-check a claim or URL            |
| POST   | `/api/check-website`  | Check a URL for safety threats       |
